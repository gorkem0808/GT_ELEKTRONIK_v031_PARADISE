BAĞLANTI LİSTESİ

PLAYER 1 MOUSE PICO:
GP26 / ADC0 = X pot orta uç
GP27 / ADC1 = Y pot orta uç
3V3 = pot bir uç
GND = pot diğer uç
GP19 = DIP switch -> GND olursa aktif
GP18 = kalibrasyon butonu -> GND basınca aktif

PLAYER 2 MOUSE PICO:
GP26 / ADC0 = X pot orta uç
GP27 / ADC1 = Y pot orta uç
3V3 = pot bir uç
GND = pot diğer uç
GP19 = DIP switch -> GND olursa aktif
GP18 = kalibrasyon butonu -> GND basınca aktif

CONTROLLER PICO:
GP0 -> tuş -> GND = keyboard 0
GP1 -> tuş -> GND = keyboard 1
GP2 -> tuş -> GND = keyboard 2
GP3 -> tuş -> GND = keyboard 3
GP4 -> tuş -> GND = keyboard 4
GP5 -> tuş -> GND = keyboard 5
GP6 -> tuş -> GND = keyboard 6 / P1 trigger / P1 röle
GP7 -> tuş -> GND = keyboard 7 / P2 trigger / P2 röle
GP8 -> tuş -> GND = keyboard 8 / kredi coin
GP9 -> tuş -> GND = keyboard 9 / start

RÖLE:
Controller GP26 = P1 röle sinyal
Controller GP27 = P2 röle sinyal
GND ortak olmalı.
